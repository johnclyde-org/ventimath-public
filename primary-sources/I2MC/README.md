# I2MC - Interlake Invitational Math Contest

## Contest Owner

**Interlake High School Math Club**
- Location: Bellevue, Washington

## About

The Interlake Invitational Math Contest (I2MC) is a student-run math competition hosted by Interlake High School in Bellevue, Washington.
