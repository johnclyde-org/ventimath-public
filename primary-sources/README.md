# Primary Sources

This folder contains primary source materials from various math competitions. Each subfolder is dedicated to a specific contest and includes attribution information for the contest owners.

All materials are collected with the intent of proper attribution and respect for copyright. If you represent a contest organization and have concerns, please open an issue.

## Contests

- **[ARML](./ARML/)** - American Regions Mathematics League
- **[AMC](./AMC/)** - American Mathematics Competitions
- **[Math is Cool](./MathIsCool/)** - Math is Cool competition series
- **[I2MC](./I2MC/)** - Interlake Invitational Math Contest
- **[KPMT](./KPMT/)** - Knights of Pi Math Tournament
- **[KNMC](./KNMC/)** - Krithik's Norz Math Contest
- **[NIMO](./NIMO/)** - National Internet Math Olympiad
