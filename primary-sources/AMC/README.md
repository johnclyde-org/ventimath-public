# AMC - American Mathematics Competitions

## Contest Owner

**Mathematical Association of America (MAA)**
- Website: https://www.maa.org/math-competitions
- AMC: https://amc-reg.maa.org/

## About

The AMC (American Mathematics Competitions) is a series of competitions including AMC 8, AMC 10, AMC 12, AIME, and USAMO/USAJMO. These competitions form the pathway to selecting the USA International Mathematical Olympiad team.
